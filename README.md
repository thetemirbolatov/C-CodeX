# C++ CodeX 1.0

**Профессиональный редактор кода для C++ с компиляцией и запуском**

---

## О программе

C++ CodeX — это легкий и мощный редактор кода, созданный специально для языка C++. Сочетает простоту Блокнота с возможностями профессиональных сред разработки. Подсветка синтаксиса, встроенная компиляция через MinGW, запуск программ и многоязычный интерфейс делают его удобным инструментом для обучения и повседневной работы.

## Возможности

### Подсветка синтаксиса C++
Полная подсветка синтаксиса языка C++. Ключевые слова, типы данных, строки, комментарии, числа, директивы препроцессора и вызовы функций отображаются разными цветами. Поддерживаются однострочные и многострочные комментарии.

### Встроенная компиляция и запуск
Компилируйте и запускайте программы прямо из редактора. Компилятор MinGW встроен в установщик. Панель вывода показывает процесс компиляции, ошибки и результат выполнения программы. Остановить выполнение можно в любой момент.

### Многоязычный интерфейс
Поддержка трех языков интерфейса:
- Русский
- Английский
- Карачаевский

Язык переключается в настройках и сохраняется автоматически.

### Нумерация строк
Каждая строка пронумерована для удобной навигации. Текущая строка подсвечивается.

### Автоотступы
Редактор автоматически сохраняет уровень отступа при создании новой строки. После открывающих фигурных скобок отступ автоматически увеличивается.

### Работа с выделенным текстом
Выделите несколько строк и нажмите Tab для добавления отступов. Shift и Tab убирает отступы. Выделение сохраняется после каждой операции.

### Управление шрифтом
Изменяйте размер шрифта через Ctrl и колесико мыши или через меню. В настройках можно выбрать любой моноширинный шрифт, установленный в системе.

### Панель вывода
Результаты компиляции и выполнения программ отображаются в отдельной панели под редактором. Ошибки выделяются красным цветом.

### Справочник синтаксиса C++
В меню Справка доступен полный справочник синтаксиса с примерами: Hello World, переменные, ввод и вывод, условия, циклы, функции, классы, векторы, указатели, шаблоны.

### Приветственное окно
При первом запуске показывается окно с описанием всех возможностей программы. Можно отключить повторный показ.

### Темная тема
Профессиональная темная цветовая схема для комфортной работы в течение длительного времени.

## Системные требования

- Windows 7, 8, 10 или 11 (32-бит или 64-бит)
- 2 ГБ оперативной памяти
- 500 МБ свободного места на диске

## Установка

### Установщик
Скачайте установщик `CppCodeX_Setup.exe`, запустите и следуйте инструкциям. Программа установится со всеми необходимыми файлами, включая компилятор MinGW.

### Портативная версия
Скачайте архив, распакуйте в любое удобное место и запустите `CppCodeX.exe`. Для компиляции программ разместите папку `mingw32` с компилятором рядом с исполняемым файлом.

## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| Ctrl и N | Новый файл |
| Ctrl и O | Открыть файл |
| Ctrl и S | Сохранить файл |
| Ctrl и Shift и S | Сохранить файл как |
| Ctrl и Z | Отменить действие |
| Ctrl и Y | Повторить действие |
| Ctrl и F | Найти и заменить |
| F5 | Компилировать и запустить |
| Ctrl и C | Остановить выполнение |
| F12 | Показать или скрыть панель вывода |
| Ctrl и Плюс | Увеличить шрифт |
| Ctrl и Минус | Уменьшить шрифт |
| Ctrl и колесико мыши | Изменить размер шрифта |
| Tab | Добавить отступ |
| Shift и Tab | Убрать отступ |
| Alt и F4 | Выйти из приложения |

## Настройки

Откройте настройки через меню. Доступные параметры:
- Язык интерфейса
- Шрифт редактора
- Размер шрифта
- Перенос длинных строк

Все настройки сохраняются автоматически.
